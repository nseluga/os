# NBA Shot Value

**Repo:** https://github.com/nseluga/Shot-Value-Machine-Learning  
**Status:** active  
**Collaborators:** Nate Seluga + Michael O'Brien (Harvey Mudd CS Dept)

One-line purpose: Predicting the expected point value of NBA shot attempts from play-by-play context (location, defender proximity, game conditions) using multiple ML approaches.

## Tech stack

- Python / Jupyter Notebook
- Models: logistic regression, tree-based ensembles, neural networks
- Interpretability: SHAP feature importance
- Evaluation: accuracy, precision/recall, F1, ROC-AUC, calibration plots

## What it does

- Engineers basketball-specific features: shot distance, defender proximity, spatial court position
- Compares multiple model families with full evaluation suite
- Generates shot value heatmaps across court zones
- Full writeup in `Project_Writeup.pdf`
