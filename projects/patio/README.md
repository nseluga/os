# Patio

**Repo:** `~/Downloads/Patio`  
**Status:** active

One-line purpose: Full-stack social betting app for backyard games (Caps, Beerball, Pong, Campus Golf) — place bets against friends (PvP) or the House (CPU), track live bets, and climb the leaderboard.

## Tech stack

| Layer | Tech |
|---|---|
| Frontend | React 19, React Router, Axios, Lucide React |
| Backend | Python Flask, JWT, SQLAlchemy |
| Database | Supabase (Postgres) |
| Deployment | Vercel (frontend) + Render (backend) |

## Features

- JWT-based auth (login & registration)
- PvP bets — create and accept bets against friends
- CPU/House bets — automated odds with ~4% house edge
- Live bet tracking — enter stats, confirm results
- Leaderboard and per-user profile with bet history
